# A-Level vs AP vs IBDP

- A-Level：英国、新加坡、加拿大、澳洲
- AP：美国、加拿大
- IBDP：除本国教育体系之外，国际上最认可的高中课程

# PSLE OLEVEL ALEVEL 
是新加坡三个国家考试：小升初，中考，高考
