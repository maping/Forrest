作业分析：
*Picture Description
我们课上讲了Forrest第二次交的图片描述。总体来说，比第一次写得更美，描述也照顾到了很多细节。
1. 颜色描述：英文里面一般不会用两个连续的颜色，而是以某个颜色为基调，加上“偏一点”另外一种颜色来描述。和中文很类似，比如“红棕色”是偏红的棕色。Forrest可以用“颜色+ish+颜色”来描述。
2. 动词用法：Forrest有尝试用一些比较小众的词汇，比如“moor”，但是要注意主被动的用法。
3. 形容词构词法：Forrest用形容词构词法“创造”了“calmful”，但是他忘记了calm本身就可以是形容词，我们常说的“keep calm”，就是形容词用法。Forrest运用了这个构词法，超级棒的，但是用错了！
4. 逗号不能连接两个完整句子：这个问题反复出现。其实Forrest分析一下句子成分，就知道是不是完整句子了。课上也和Forrest一起将句子修改了一下，比如直接断开变成两个单独的句子，或者用逻辑连接词将前后构成逻辑关系等等。
5. 这次图片描述最大的问题就是Forrest没有加入方位引导。想象一下，如果读者没看到图片，只能感受到这是很美的画面，但是很难想象出一幅构图不错的画面。课上问Forrest这句话是否清楚：在老师共享的界面里面，格式刷的右边是粘贴。
Forrest很难找出格式刷在哪。然后我们换了一个说法：“在共享界面的左上角有一个格式刷，格式刷的右边是粘贴。”Forrest很快就找到了。所以方位非常重要，也是让图片描述变得很有逻辑的方法。Forrest可以先给读者在图片里定位，已经定位的物体，就可以作为其他东西的方位参考了。

*Article
第二次交的作文比第一次好很多啦！！但是我们还是两篇作文都点评了一下，很多问题是Forrest反复出现的，或者很值得分析的。
Introduction：
1. 抄串行了：这已经不是一次两次啦！之前Forrest说写完会重新再看一遍，一定要坚持做这些看似没必要的工作。抄也可能抄错的，更何况考试的时候是“抄”脑子里想的呢！
2. 形近词用错：option和opinion很像，但Forrest肯定认识！这和第一点其实一样，还是不细心，而不是Forrest区分不了这两个单词。
3. 第一次写的article中，自己的观点和下文分析的内容不对等。比如开头最后一句提到“independence和confidence”，但是body paragraph的第一句，写的是responsibility。所以不建议Forrest现在写了自己观点之后解释原因。因为原因正是两个body paragraph要分析的内容，Forrest很容易写偏。
4. 第二次写的article没有写到Forrest的观点，结构不完整。
Body Paragraph 1:
1. hedging：Forrest写作文语气要委婉，不要太肯定，比如“学生忘记带作业”，一定会面临某种后果吗？很大程度上是取决于不同老师对这件事的处理方式。所以Forrest最好说“可能会面临xx后果”。
2. 第一篇作文Forrest说忘记带作业会有“embarrassment”，但是“忘带作业”和“embarrassment”之间是经过很多环节的。比如：老师直接在课上点名了，大家都听到了，所以没带作业的学生觉得尴尬。这可能是很多大陆的老师的做法，老外可能就不明白为什么会尴尬了。所以要加以描述。
很多时候，Forrest的脑回路太快了，这也是之前说Forrest写得不够详细的原因。和理科分析跳步骤很像呀。

其他：
有一个很重要的问题是，firstly和secondly不能用在持“balance”态度的作文上。因为firstly和secondly是分别引出支持同一个观点的内容；但是balance是两个对立的概念里面找balance。所以要用“however”、“on the one hand… on the other hand”形成转折。

课堂作文练习：
今天我们分析作业的时候，也一起和Forrest回顾了之前讲的图片描述方法和article结构，我们留给新作文的时间没有很多。但是，我们一起探讨了可能的思路，以及如何分析某个论点。
1. Forrest的思维其实很发散，而且点子很新颖。比如“机器人的运用在某种程度上会增加就业，因为需要机器人维修工”，这个点子真的很棒，因为大多数人想到的都是机器人取代人，而Forrest两方面都想到了！
2. Forrest现在会联系到自己的见闻，反推论点；而不是站在一个很抽象的立场去使劲儿想论点，再挑例子对号入座。比如课上提到了给F1赛车换轮胎，也是用机器人，Forrest就推出一个论点：机器人维护费用很高。
这也是一个很大的进步！！
