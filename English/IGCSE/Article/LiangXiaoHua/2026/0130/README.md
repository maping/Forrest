作业分析：
*Picture Description
这次图片描述整体写得很好，Forrest能按照方位顺序描写看到的东西，形容词也用得比较恰当。
1. 图片描述的最开头要有总起句，告诉读者这是一张关于什么的图片。课上给了Forrest一些公式：This is a picture/photo of + 地点/主题。
2. 关于颜色的形容词，Forrest和之前说的连字符（hyphen）构词法混淆啦。如果是很多形容词叠在一起，是有顺序规律的（英语形容词顺序规则/Adjective Order）这点Forrest已经形成了语感。课上我们讲了在什么情况下可以用连字符自创一些单词。
3. 有些词比较口语，不符合Forrest整篇图片描述的风格，比如“feeling”、“look”等，可以换成“atmosphere”和“appearance”。

*Article
Article的分析逻辑是很不错的，但是在其他地方出现了小问题：
1. introduction的background没写。课上问Forrest，他说background是“In recent class discussion”。这是不能作为background的哦，不然我们不需要专门练习啦，每篇文章都可以用它作为background？！NO！background是关于这个话题内容的，要告诉读者为什么会有这个（课堂）讨论，关于这个讨论，现状是怎么样的。比如这次的话题是“food waste”。课上，我们重新写了background，并和Forrest一起总结了“公式”。
2. Forrest把第一个论点分析得很好，但是结构需要调整。而且，Forrest写的例子不能当例子，而是一个分析。
课上我们调整了顺序，变成：
a 临期食品要扔掉以保护顾客健康
b 解释为什么这么做（会生病）
c 解释为什么会生病
d 总结
之前Forrest的顺序是abdc。调整过后，Forrest缺少一个例子，课上我们一起写了。Forrest的思路很好，给出了很具体的画面。
对比起以前的article，Forrest会发现，老师的要求高了很多，已经不再满足于Forrest一句话的分析和例子了。因为，老师想Forrest尽量多写，做到细致和缜密。EFL要求这么多字数，Forrest要慢慢适应。
3. 原来Forrest一直不知道自己拼错government！歪个楼，回到free talk，很多Forrest的错误表达、错误拼写等，已经形成了习惯，要不断提醒改正。如果课上指出，复习的时候，就得格外注意啦！
4. 有时候Forrest自己写偏了都没发现呢。但这次作文挺好的，就只有一个词是无关的，已经进步啦，不会像之前，有几句话是和上下文无关的。“While throwing away food seems wasteful, the potential cost of human illness and legal liability is far greater.” 这句话看似没问题，但是“legal liability”和本段论点的内容“customers’ health”无关！和论点2有关。
5. Forrest在论点2提到food bank，Forrest要稍微解释一下它是什么，或者它在food waste上做了什么，不然读者会觉得很抽象。另外，虽然Forrest翻译的是“食物银行”，但其实bank也有“库”的意思哦！
6. 论点2的例子也属于分析，课上一起想了一个新例子，并且引导Forrest怎么写得具体和有画面感。

课堂作文练习：
今天讲的话题恰好和Forrest在闲聊的时候分享的事情联系上了。Forrest想法挺多的。这次课我们主要是讨论正反方的观点，以及分析原因。
1. Forrest在写作的时候，很喜欢结合自己的经历，这很好。但是记得要“generalize”，写成大家都有可能经历的事情，否则，读者会觉得和自己无关，很有距离感。比如，Forrest提到，他们学校放学后很吵，不适合安静地写作业。但是老师反驳，如果大家都写作业，怎么会吵呢？所以，Forrest首先不要说是自己学校，而是说“有的学校”，并加以合理的解释（有学生参加课后校内活动？体育训练？音乐排练等）。
2. Forrest想另一方观点的时候，只需要反驳对方就有很多idea了。
