Free Talk:
课上先问了一下Forrest回家过得怎么样，他的回答是“挺充实的”，我们由此展开了图片描述的话题。我们先回答了问题：到底是如何充实呢？然后开始讨论了Forrest的期末考试图片。
课上发现，Forrest说出来的东西都是很基本的，看到什么就说什么，没有修饰，也不按方位逻辑。比如：邮轮、游乐场、滑滑梯、餐厅……。
那邮轮在哪里呢？邮轮上的设施的布局怎么样？什么颜色？占地多大？等等很多细节都没有说到。图片描述的关键是，让别人能从描述中脑补出一幅很丰富的画面。

看图说词：
这本来是一个三秒看图说词的游戏，不过我们没有倒计时，不给Forrest压力了，他有充足的时间思考。
Forrest需要更多丰富的词汇，特别是形容词，用于详细描述眼前的东西。我们先玩了个游戏热热身。Forrest喜欢用一些概括性很强的词，但是不够具体，比如形容气味：smells strong，形容食物变坏：gets bad等等。

图片描述：
听了Forrest描述期末考的邮轮图片，我们就知道Forrest在图片描述上，需要扎实什么功底。课上我们回顾了讲故事最常用的5W1H，重新梳理邮轮图片中，Forrest看到的东西。Forrest可以从最吸引眼球的东西说起，然后根据它的位置，从上到下、从下到上、从左到右，从右到左，从中间到四周……都可以。先把大致的内容描述一遍，再根据方位，讲细节。
Forrest练起来还是有点吃力的，可能以前都是看到什么说什么，现在要开始加入逻辑了，还要求有很强语言表达能力。

作业分析：
*Summary
Forrest的Summary虽然都总结到位了，但是最近的内容也越来越长了呢。比如说有比较多时间状语从句、定语从句等等。写article的时候，非常鼓励Forrest写从句哦！但是Summary要精简。

*Article
Article的内容是很精彩的，但是论述段的结构有点乱。
1. 开头段的结构超棒的，之前课上强调的内容，都写到了！不过拼写需要注意，特别是题目有的单词，更加不能错了！（safety）
2. 论点不宜过长。第一个论点非常长，而且包含很多细节内容。论点是需要言简意赅的。这次论点包含的例子，应该写在论点之后。
3. 举例子不连贯。因为Forrest在论点写了例子，后面也有例子，所以显得比较分散，而不是一个很有逻辑的顺序：论点 - 分析 - 例子 - 论点扣题。课上我们重新“组装”了这个段落，基本内容都没变，甚至都是用Forrest原有的，改掉语法和用词的句子。所以，Forrest的内容是丰富的，只不过有点乱。
4. 语言准确度要提高。用词需要更加正式，比如know，在Forrest原句中，能代替的词太多啦（realize、understand等等）；hedging也是要用上的。不要说某件事必然会导致一个结果，而要说“很大程度上”、“可能”、“大部分人会……”，这种比较委婉的表达方式。
